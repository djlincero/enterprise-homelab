# Changelog

All notable changes to this project will be documented in this file.

This project follows a milestone-based development process.

## v0.1.0 – Infrastructure Foundation

### Added

* Proxmox VE virtualization platform
* pfSense firewall
* Enterprise VLAN segmentation
* Internal routing
* Windows Server 2022 virtual machines
* Windows 11 client
* Debian administration server

### Added

* Active Directory Domain Services
* DNS
* DHCP
* Enterprise Certificate Authority (PKI)
* File Server

## v0.3.0 – Infrastructure Automation

### Added

* Ansible Controller
* SSH automation
* WinRM automation
* Linux baseline playbook

* Connectivity testing
* Fact gathering playbooks
* Site-wide deployment using `site.yml`
