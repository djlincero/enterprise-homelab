# Wazuh Dashboard

## Objective

Validate that the Wazuh Dashboard provides centralized visibility of the entire enterprise infrastructure.

---

# Dashboard Features

The Wazuh Dashboard provides a single pane of glass for monitoring security events generated by all managed systems.

Available features include:

* Agent monitoring
* Security events
* Inventory
* Vulnerability Detection
* File Integrity Monitoring
* Security Configuration Assessment
* Threat Hunting
* Dashboard Visualizations

---

# Agent Overview

The dashboard displays:

* Agent Name
* Agent ID
* Operating System
* Agent Version
* Node Assignment
* Current Status
* Last Keep Alive
* IP Address

The following systems are monitored:

* Windows 11
* Domain Controller 01
* Domain Controller 02
* Certificate Authority
* File Server
* Debian Linux
